# Art Placeholder Notes

The current prototype draws the game with runtime shapes so it can run before final resources exist.

## Recommended Theme Swaps

- village core: bell tower, granary, shrine, or town hall
- arrow tower: watchtower
- cannon tower: stone thrower
- frost tower: well, shrine, or ice totem
- enemies: wolves, boars, bandits, siege carts

## Asset Priority

1. Core building
2. Three towers
3. Three enemy silhouettes
4. Ground tiles
5. Hit effects
6. HUD buttons

## Safe Reuse Guidance

Do not directly reuse art, audio, or UI from the legacy carrot project.
It is fine to borrow:

- map layout ideas
- wave pacing ideas
- tower role ideas
- data structure ideas
